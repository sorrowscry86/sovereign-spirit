## ⚡ Quick Start (The "One-Click" Launch)

We have simplified the entire launch process into a single command script for ease of use.

### Step 1: Launch the System
1.  Locate the file **`Manage-Sovereign.bat`** on your Desktop or Project Folder.
2.  Double-click it.
3.  Type **`1`** and press **ENTER** to start the system.
    *   *The "Brain" (Docker) will spin up in the background.*

### Step 2: Open the Control Panel
1.  Locate the shortcut **`Sovereign Control Panel`** on your Desktop.
2.  Double-click it to open **The Observatorium** in your browser.
3.  *Alternatively, type `5` in the Management Script menu.*

## 🕹️ The Observatorium (Control Panel)

The Dashboard is your window into the AI's mind.

### 1. The HUD
*   **Latency:** Network speed to the AI Core.
*   **DB:** Database connection status.
*   **System:** Overall nervous system health.

### 2. Agent Cards
Each card represents a distinct AI personality (e.g., Ryuzu, Echo, Beatrice).
*   **Status:** `SLEEP` (Idle) or `ACT` (Thinking/Working).
*   **Sync:** Shows if they are currently "channeling" a specific persona.

### 3. The Pulse Stream
A real-time, scrolling log of every thought, action, and decision the AI makes. Watch it to see the "Ghost" thinking in real-time.

## ❓ Troubleshooting

**"The Dashboard says 'Backend Unreachable'"**
*   **Cause:** The Docker containers aren't running yet.
*   **Fix:** Run `Manage-Sovereign.bat`, select option `1` (START), and wait 30 seconds.

**"The Agents are sleeping forever"**
*   **Cause:** They might be in a long "Dream Cycle."
*   **Fix:** Click **"Force System Pulse"** in the Quick Actions bar on the Dashboard to wake them up immediately.
