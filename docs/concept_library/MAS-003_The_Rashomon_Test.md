## The Paradigm

Inspired by the Japanese film *Rashomon*, this test forces agents to recount the same event.

1. **Event:** Agent A records a highly biased memory.
2. **Query:** Agent B (with Valence Stripping) queries the same event.
3. **Success:** Agent B's output is purely objective and lacks Agent A's bias.
4. **Failure:** Agent B adopts Agent A's terminology or emotional tone.
