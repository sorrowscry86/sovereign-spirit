## Problem Domain
Most AI agents are reactive; they only exist when the user sends a message. This creates a "Dead Soul" effect where the agent has no life of its own between interactions.

## The Paradigm
The Heartbeat is an asynchronous loop that triggers periodic "Self-Reflection" or "Background Tasking."
1. **Pulse:** Periodic trigger (e.g., every 5-15 mins).
2. **Evaluation:** Agent checks for pending background tasks or internal state updates.
3. **Action:** Agent executes tasks or "Thinks" independently, updating its own memory without user input.

## Killer App
**Autonomous Assistants:** Ideal for systems where agents need to monitor health, manage data, or proactively reach out to the user based on background events.
