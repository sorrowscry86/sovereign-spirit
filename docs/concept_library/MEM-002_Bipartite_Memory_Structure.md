## Problem Domain
Flat memory structures (Text + Metadata) don't distinguish between the "What" (Facts) and the "How it felt" (Subjectivity), making it impossible to share data while preserving perspective.

## The Paradigm
Every memory object contains two primary fields:
1. **Objective Fact:** The immutable, shared truth of the event.
2. **Subjective Voice:** The author's unique emotional coloring and interpretation.

By separating these at the schema level, the system can selectively expose or hide the subjective layer based on the retrieving agent's identity.

## Killer App
**Shared Knowledge Bases:** Perfect for multi-agent worlds where agents need to share knowledge without losing their individual "souls."
